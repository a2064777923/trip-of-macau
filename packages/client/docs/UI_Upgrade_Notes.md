# 澳小遊 UI 升級與功能迭代說明

在這一次的開發與迭代中，我們專注於提升應用程式的美觀程度及遊戲化體驗。以下是具體的更新說明：

## 1. 首頁 - 旅程評估彈窗與 AI 加載過渡
為了讓使用者在初次使用或來到新城市時的體驗更加流暢且具儀式感，我們將原先嵌入在首頁的「旅程評估」模組升級為**模態彈窗 (Modal)**。
- 透過半透明遮罩背景，讓使用者專注於設定當天的遊玩偏好。
- 在點擊「生成推薦」後，加入了一段 **AI 規劃中的加載動畫與文案**（持續 1.5 秒），讓使用者感受到系統正在為其量身打造專屬內容的過程。
- 同時，評估選項的排版也進行了微調，使介面看起來更緊湊美觀。

## 2. 首頁 - 全新城市地圖切換功能
為了支援多個城市的探索，我們在首頁英雄區域 (Hero Section) 的右上角加入了**切換地圖入口**。
- 點擊地圖圖示後，會彈出一個**橫向滑動的城市選擇器**。
- 每個城市以圖文卡片呈現（預留了放置手繪風格封面的空間），並標示了該城市的解鎖狀態。
- 新增了「華東師範大學」的校園怪談地圖，並開放了選擇與進度展示。
- 尚未抵達或未解鎖的城市會呈現半透明狀態並附有鎖頭圖示，點擊時會提示無法切換。
- 切換城市後，首頁資訊（包含標題、印章與進度等）將會自動刷新。

## 3. 地圖頁 - 全屏遊玩與區域限制
為了強化「探索」的沉浸感與遊戲性，地圖頁進行了重要的交互升級。
- **初始定位與縮放**：地圖預設會縮放至最大等級 (scale 18)，讓使用者感覺更貼近實際街道。
- **定位精確度與使用者圖示**：我們在地圖上增加了自定義的「小魚」使用者圖標 (`user.png`) 來替代原本的藍點，頭部方向會隨手機方向調整。
- **區域限制 (Region Bounds)**：系統會即時檢測使用者的經緯度。若使用者不在目前選擇的城市（如澳門、華東師大）的行政區域內，地圖會被鎖定在城市中心點，並覆蓋一層提示遮罩：「您不在這個區域內，沒法探索呢~」。
- **POI卡通圖示與互動升級**：我們為澳門的著名景點（如大三巴、玫瑰堂、葡京人等）和華東師大的景點（如麗娃河、文史樓）加入了專屬手繪風格圖標（如 `ruins.png`, `ghost.png` 等）。當使用者進入 50 米範圍內並點擊圖標時：
  1. 系統將會以 Bottom Sheet （氣泡選擇單）的形式彈出。
  2. 使用者可以選擇：打開室內地圖（佔位）、查看達人推薦（跳轉至秘籍頁，已修正為正確的 `switchTab` 跳轉方式）、探索故事線索（跳轉至對應故事線）。

## 5. 首頁 - 修正
- 修復了點擊推薦內容後的跳轉報錯問題，對於 tabBar 頁面（如「秘籍」），現在已改用正確的 `Taro.switchTab` 方法。
- 首頁在完成「旅程評估」AI 加載後，現在會正常顯示生成的推薦路線、原因和推薦的圖文內容，並且改為由用戶主動點擊「太棒了，開始探索！」後才關閉彈窗。
- 修正了地圖圖標的顯示層級與定位邏輯，使「小魚」頭像和著名景點（POI）能共存並正常互動。

## 6. 全新室內地圖系統 (Indoor Map)
為大型商場或地標（如：葡京人 Lisboeta）加入了專屬的室內地圖功能。
- **瓦片地圖渲染**：讀取生成的瓦片圖檔 (Tiles) 渲染各樓層，支援 `地下 (G)`、`一樓 (1F)`、`二樓 (2F)`。
- **手勢互動**：支援在手機端進行拖曳 (Drag) 以瀏覽地圖各處。
- **縮放功能**：側邊工具列提供了 `+` / `-` 按鈕供使用者放大、縮小地圖視角。
- **AR 定位 (模擬)**：加入了 AR 視覺定位的互動設計。使用者點擊「AR定位」後會出現相機掃描的模擬過渡動畫。定位成功後，地圖會自動切換至對應樓層、將視角置中，並放置一個附有呼吸燈效果及方向指引光束 (Cone) 的使用者標記。方向光束會與手機羅盤 (Compass) 實時同步。
- **AI 導覽入口**：在地圖工具列預留了「AI 對話」的語音對話按鈕。

## 4. 底部導航列 (Tab Bar) - 圖示升級
為了擺脫系統預設圖示的生硬感，我們為底部導航列設計了 5 組簡約優雅的向量圖示（包含未選中與選中狀態）：
- 🏠 **首頁** (Home)
- 📍 **探索** (Map)
- 🧭 **發現** (Discover)
- 📝 **秘籍** (Tips)
- 👤 **我的** (Profile)
由於微信小程序原生的 `tabBar.list[].iconPath` 配置項不支援 `.svg` 格式，我們將所有設計好的 SVG 轉換為 `64x64` 大小的 PNG 檔案（支援格式）。圖示採用了柔和的灰藍色 (`#A0AEC0`) 與品牌主色調櫻花粉 (`#FFB6C1`)，並在 `app.config.ts` 中完成了設定，使整體風格更具現代感與質感，不再有明顯的「AI 生成感」。

## 5. UI 細節微調
- 引入了 `logo.png` (魚小遊) 作為應用程式的標誌，放置於首頁標題旁，增添親切感。
- 調整了首頁 Hero 區塊的排版，使其能更好地容納標誌與地圖切換按鈕。
