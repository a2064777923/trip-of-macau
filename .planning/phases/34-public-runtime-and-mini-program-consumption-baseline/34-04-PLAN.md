  - `storyModeConfig` exists

  - repeat the same event call and assert the duplicate returns the same event id or same client event id

  - call `POST /api/v1/storylines/{id}/sessions/{sessionId}/exit`
- print exactly `Phase 34 public runtime smoke passed` on success

Do not hardcode bearer tokens, usernames, passwords, WeChat secrets, COS secrets, or API keys.
