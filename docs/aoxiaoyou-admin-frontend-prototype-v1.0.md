## 一、设计规范

### 1.1 设计原则
- **清晰**: 信息层级分明，一眼找到重点
- **高效**: 常用操作不超过3步完成
- **一致**: 相同功能用相同交互模式
- **反馈**: 操作后有明确状态反馈

### 1.2 色彩规范

```css
/* 主色调 - 澳门特色蓝 */
--primary: #1890ff;
--primary-light: #40a9ff;
--primary-dark: #096dd9;

/* 功能色 */
--success: #52c41a;   /* 成功 */
--warning: #faad14;   /* 警告 */
--error: #f5222d;     /* 错误 */
--info: #1890ff;      /* 信息 */

/* 中性色 */
--text-primary: rgba(0, 0, 0, 0.85);
--text-secondary: rgba(0, 0, 0, 0.45);
--border: #d9d9d9;
--background: #f5f5f5;
```

### 1.3 字体规范

```css
/* 中文字体 */
font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', sans-serif;
```

```typescript
// 验证规则示例
const validationRules = {
  poiName: [
    { required: true, message: 'POI名称不能为空' },
    { max: 128, message: '名称最多128个字符' }
  ],
  coordinates: [
    { required: true, message: '坐标不能为空' },
    { 
      validator: (value: string) => {
        const [lat, lng] = value.split(',').map(Number);
        return lat >= 22.1 && lat <= 22.2 && lng >= 113.5 && lng <= 113.6;
      },
      message: '坐标必须在澳门范围内'
    }
  ]
};
```

```typescript
// 路由懒加载
const POIManagement = React.lazy(() => import('./pages/POI'));
const UserManagement = React.lazy(() => import('./pages/User'));

// 组件懒加载
const MapComponent = React.lazy(() => import('./components/Map'));
```

### 7.2 数据请求优化

```typescript
// React Query 配置
const queryConfig = {
  staleTime: 5 * 60 * 1000,    // 5分钟内数据视为新鲜
```
