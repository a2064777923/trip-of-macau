# 東西方文明的戰火與共生

## 故事線總覽

玩家身份是穿越時空的「濠江歷史見證者」。故事以一枚殘缺的海防銅鏡為主線信物，串聯媽閣廟、亞婆井前地、崗頂前地、大炮台與議事亭前地五個地點。這條故事線不是把每個 POI 當作孤立景點，而是把 POI 預設體驗、故事模式覆寫、支線拾取、隱藏挑戰、獎勵稱號與探索元素整合成一條可編排的路線。

探索度使用 exploration_elements 權重動態計算，不在流程中寫死百分比。

## 第一章：鏡海初戰：中葡首次海防對峙

- 錨點 POI：媽閣廟，`ama_temple`
- 史實依據：16 世紀中葉葡萄牙人逐步進入澳門並形成貿易與居留活動，媽閣一帶可作為海岸信仰、航海記憶與早期中葡接觸的入口。
- 文學演繹：以「鏡海初戰」重構海岸防務、漁民防線與武裝商船對峙；銅鏡碎片與三個主線疊加物屬故事化設計。
- 主線互動：抵達 50 米範圍後播放核心劇情短片與古戰場音頻；依次點擊「明朝水師戰船」「媽閣漁民防線」「葡國武裝商船」三個疊加物後通關。

- 探索權重：主線與隱藏挑戰為 `core`，支線拾取為 `large`，媒體閱讀為 `medium`。

## 第二章：南灣防線：葡人築城的邊界博弈

## 第三章：山城戒備：教會與軍防的雙重佈局

- 錨點 POI：崗頂前地/崗頂劇院一帶，優先 `st_augustine_square`

## 第四章：炮台硝煙：荷澳戰役的生死反擊

- 史實依據：1622 年荷蘭艦隊攻澳與大炮台防守是澳門軍事史重要節點。

## 第五章：烽煙落幕：從對峙到文明共生

- 史實依據：澳門長期形成中西建築、宗教、商貿與城市治理交疊的文化景觀，可作為文明共生終章。
