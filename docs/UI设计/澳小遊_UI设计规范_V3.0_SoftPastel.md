# 澳小遊 UI 设计规范 V3.0 - Soft Pastel 风格

## 项目信息
- **项目名称**：澳小遊 (Trip of Macau)
- **风格方向**：Soft Pastel（柔和粉彩）
- **版本**：V3.0
- **设计师**：玖妖丸

---

## 一、设计理念

### 1.1 风格定位
- **柔和治愈**：使用低饱和度粉彩色系，营造温馨、友好的氛围
- **清新自然**：避免强烈对比，追求视觉舒适度
- **现代简约**：保持界面整洁，信息层级清晰

### 1.2 设计原则
1. **一致性**：全应用使用统一的色彩、字体、圆角、阴影
2. **可读性**：确保文字与背景有足够对比度
3. **触感反馈**：所有可交互元素提供视觉反馈
4. **无障碍**：支持长者模式，字体可放大

| Token | 值 | 用途 |

## 六、组件规范

### 6.1 按钮

#### 主按钮（Primary）
- 背景：`pastelPink`
- 文字：`#FFFFFF`
- 圆角：`radius-full`
- 高度：48px
- 内边距：0 24px

#### 次要按钮（Secondary）
- 背景：`pastelBlue` opacity 0.2
- 文字：`textPrimary`
- 圆角：`radius-full`
- 高度：40px
- 内边距：0 20px

#### 长者模式按钮
- 最小高度：64px
- 字体大小：20px
- 圆角：16px
- 内边距：0 32px

### 6.2 卡片

#### 标准卡片
- 背景：`cardBg`
- 圆角：`radius-lg`
- 阴影：`shadow-md`
- 内边距：`space-md`

#### 图片卡片
- 图片圆角：`radius-lg`
- 内容区内边距：`space-md`
- 标签位置：图片左上角

- 圆角：20px

- 点击区域扩大

### 6.3 导航栏

### 6.4 输入框

- 背景：`#F7FAFC`
- 圆角：`radius-md`

- 聚焦边框：`pastelBlue`

- 占位符颜色：`textTertiary`

## 七、页面布局规范

### 7.1 页面边距
- 标准边距：16px（左右）
- 长者模式：20px（左右）
- 大屏适配：最大宽度 430px，居中显示

### 7.2 内容区
- 标准卡片间距：16px
- 列表项间距：12px
- 大区块间距：24px

### 7.3 安全区
- 顶部安全区：44px（状态栏）
- 底部安全区：22px（iPhone X+）
- 内容避开安全区

---

## 八、长者模式规范

### 8.1 视觉增强
- 全局字体放大 1.5 倍
- 最小触摸目标 64×64px
- 按钮高度最低 64px
- 对比度增强

### 8.2 交互简化
- 每屏只展示一个主要操作
- 复杂流程分步骤展示
- 自动语音播报（可选）
- 操作二次确认

### 8.3 功能精简
- 只保留核心功能
- 隐藏高级选项
- 一键求助入口常驻
- 紧急联系人快速拨打

---

## 九、适配规范

### 9.1 屏幕适配
- 设计基准：375×812px（iPhone X）
- 最小支持：320×568px（iPhone SE）
- 最大适配：428×926px（iPhone 12 Pro Max）

### 9.2 缩放规则
- 宽度 < 375px：内容等比缩放，边距自适应
- 宽度 > 375px：内容居中，最大宽度 430px
- 高度变化：内容区可滚动，固定关键元素

### 9.3 安全区适配
- 使用 `safe-area-inset` 常量
- 底部导航避开 Home Indicator
- 顶部内容避开刘海/灵动岛

## 十、图标规范

### 10.1 图标风格
- 线性图标：2px 描边，圆角端点
- 面性图标：填充式，圆角处理
- 颜色：遵循色彩系统，支持主题切换

### 10.2 图标尺寸
- 导航图标：24×24px
- 按钮图标：20×20px
- 列表图标：16×16px
- 装饰图标：自定义

### 10.3 Emoji 使用
- 作为图标补充，增强情感化
- 遵循系统 Emoji 样式
- 大小与文字匹配，适当放大

---

## 十一、动画规范

### 11.1 动画原则
- 有目的：动画服务于功能和情感
- 有节奏：缓动自然，时长适中
- 有节制：不过度使用，避免干扰

### 11.2 动画时长
- 微交互：150-200ms
- 页面切换：300-400ms
- 复杂动画：500-600ms

### 11.3 缓动函数
- 标准：`cubic-bezier(0.4, 0, 0.2, 1)`
- 减速：`cubic-bezier(0, 0, 0.2, 1)`
- 加速：`cubic-bezier(0.4, 0, 1, 1)`

## 十二、文件命名规范

### 12.1 设计文件命名
```
澳小遊_V{版本号}_{风格}_{页面名称}.{格式}

示例：
- 澳小遊_V3_SoftPastel_首页.svg
- 澳小遊_V3_SoftPastel_地图页.svg
- 澳小遊_V3_SoftPastel_到达触发.svg
```

### 12.2 组件命名
```
{类型}_{名称}_{状态}

示例：
- btn_primary_default
- card_image_hover
- input_search_focus
```

### 12.3 图层组织
```

├── 📁 01_状态栏

├── 📁 03_内容区
│   ├── 📁 01_用户信息
```
