# 澳小遊（Trip of Macau）后台管理系统技术方案
**版本：V1.1** | **日期：2026年4月8日** | **状态：已实现**

---

## 一、概述

### 1.1 项目背景
澳小遊后台管理系统是为「澳小遊」微信小程序提供的内容管理、用户管理、运营支持和系统配置的综合性管理平台。

### 1.2 设计目标
- **快速迭代**：支持内容快速更新，无需发布新版本即可上线新内容
- **数据驱动**：提供完善的数据统计和分析能力，支撑运营决策
- **测试友好**：提供完整的测试账号管理和数据调整功能
- **安全可靠**：完善的权限管理和操作审计，保障数据安全

### 1.3 技术选型原则
- **成熟稳定**：选择经过生产环境验证的技术栈
- **生态丰富**：优先选择社区活跃、文档完善的技术
- **团队熟悉**：考虑团队现有技术储备和学习成本

## 三、核心功能模块

### 3.1 模块划分

| 模块 | 功能描述 | 主要用户 | 优先级 |
|------|----------|----------|--------|
| 内容管理 (CMS) | POI管理、故事线管理、地图瓦片管理、多媒体资源管理 | 内容编辑、运营人员 | P0 |
| 用户管理 | 用户列表、用户详情、测试账号管理、数据调整 | 运营人员、测试人员 | P0 |
| 运营管理 | 活动发布、广告管理、奖励配置、数据统计 | 运营人员、市场人员 | P1 |
| 系统管理 | 账号管理、角色权限、操作日志、系统配置 | 系统管理员 | P1 |
| 测试工具 | 模拟定位、数据重置、批量操作 | 测试人员 | P0 |

### 3.2 内容管理 (CMS)
- **POI 管理**：列表、新增/编辑/删除、批量导入/导出、分类管理、地图预览
- **故事线管理**：列表、章节管理（富文本编辑）、多媒体关联、解锁条件配置
- **地图瓦片管理**：瓦片上传/预览/版本管理、CDN刷新、控制点配置
- **多媒体资源管理**：图片/音频/视频资源上传、分类管理、引用统计

### 3.3 用户管理
- **用户列表**：基本信息、高级搜索、用户详情、数据导出
- **测试账号管理**：测试账号标记/取消标记、测试数据面板

### 3.4 测试工具（核心功能）

| 功能 | 说明 |
|------|------|
| 模拟定位 | 地图点选 + 坐标输入，无需真实去澳门 |
| 印章操作 | 快速获得/批量获得/删除/清空印章 |
| 进度重置 | 重置所有游戏进度、故事线进度 |
| 等级调整 | 直接设置用户等级和头衔 |
| 触发模拟 | 模拟到达指定POI，测试弹窗逻辑 |
| 批量操作 | 对多个测试账号执行相同操作 |
| 操作日志 | 记录所有操作，支持回滚 |

### 3.5 运营管理
- **活动发布**：创建/编辑/删除，时间配置、范围配置、效果统计
- **广告管理**：广告位管理、内容管理、投放时间、A/B测试
- **奖励配置**：奖励类型/规则/库存管理、领取统计、发放记录
- **数据统计**：DAU/MAU、新增用户、留存率、印章收集、POI热度

### 3.6 系统管理
- **账号管理**：管理员CRUD、密码重置、状态管理
- **角色权限（RBAC）**：菜单权限、按钮权限、数据权限
- **操作日志**：登录日志、操作日志、数据变更日志、异常日志
- **系统配置**：基础配置、定位配置、缓存配置、第三方服务配置

## 七、安全措施

### 7.1 认证鉴权
- **JWT双Token**：Access Token（2小时）+ Refresh Token（7天）
- **密码加密**：BCrypt算法
- **登录失败锁定**：5次失败锁定30分钟

### 7.2 权限控制（RBAC）

| 角色 | 权限范围 |
|------|----------|
| 超级管理员 | 所有权限 |
| 内容运营 | CMS模块 |
| 活动运营 | 运营管理模块 |
| 测试人员 | 测试工具模块 |
| 数据分析 | 数据只读 |
