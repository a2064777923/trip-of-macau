### 1.2 路由模式

前端现已改为 `HashRouter`，在线上使用如下形式：
- `#/dashboard`
- `#/space/cities`
- `#/system/roles`

这解决了静态托管环境下 SPA 刷新和直连 404 问题。

## 五、设计原则更新

- 从“单页堆功能”升级为“域分组 + 子页承载”
- 从“原型展示”升级为“真实接口驱动页面”
- 从“局部 CRUD”升级为“面向多城市 / RBAC / 收集系统的正式底座”
