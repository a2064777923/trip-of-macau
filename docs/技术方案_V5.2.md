## 一、總體技術架構

**系統架構：**
```
客戶端層 → API網關層 → 業務服務層（Spring Boot單體）→ 數據/存儲層
```

| 層級 | 技術選型 |
|------|----------|
| 小程序框架 | Taro 3.x + React 18 |
| 地圖渲染 | Canvas 2D + 瓦片加載 |
| 狀態管理 | Zustand / Jotai |
| 動畫 | Framer Motion / 自研Canvas |
| 音頻 | InnerAudioContext |
| 定位 | wx.getLocation + 騰訊地圖SDK + Wi-Fi指紋 |
| 後端 | Spring Boot 3.x (Java 17) |
| 數據庫 | MySQL 8.0 + Redis 7.x |
| 對象存儲 | OSS + CDN |

## 二、手繪地圖系統

### 2.1 設計理念
> 先用一種卡通風格，留接口後期擴展

- 只做**一種卡通風格**手繪地圖（澳門半島核心區）
- 地圖資源**全部CDN化**，不通過小程序包下載
- 預留**風格切換接口**

### 2.2 CDN瓦片加載策略

| 資源類型 | 加載時機 | 大小 |
|----------|----------|------|
| 縮略圖 | 即時 | <100KB |
| 瓦片（Z1） | 進入頁面 | 12片 |
| 瓦片（Z2） | 按需 | 48片 |
| 控制點 | 首次+緩存 | ~50KB |
| POI數據 | 首次+緩存 | ~100KB |
| SVG元素 | POI點擊 | 按需 |

### 2.3 地圖資源CDN URL
```
cdn.tripofmacau.com/maps/macau/thumb.jpg        # 縮略圖
cdn.tripofmacau.com/maps/macau/tiles/{z}/{x}/{y}.jpg  # 瓦片
cdn.tripofmacau.com/maps/macau/geo.json         # 控制點
cdn.tripofmacau.com/maps/macau/pois.json        # POI數據
cdn.tripofmacau.com/maps/macau/svg/{id}.svg     # SVG元素
```

## 三、定位與觸發系統

### 3.1 多源定位架構

```
GPS（室外） ────→ 精度5-20米
    ├── Wi-Fi指紋（室內/信號弱）──→ 精度10-30米
    └── 手動補簽（兜底）──→ 200米內有效
```

### 3.2 動態觸發配置

| GPS精度 | 觸發半徑 | 停留時間 |
|---------|----------|----------|
| <10米 | 30米 | 30秒 |
| 10-20米 | 50米 | 60秒 |
| >20米 | 80米 | 120秒 |

**其他配置：** GPS採樣間隔2秒，冷卻時間30分鐘，防抖延遲2秒

### 3.3 到達觸發狀態機
```
IDLE → APPROACHING → DWELLING → TRIGGERED → COOLDOWN（30分鐘）
```
