## 執行摘要

澳小遊是全球首個「卡通手繪地圖 + 故事解謎 + 遊戲化探索」的澳門文旅小程序。

**核心差異化亮點：**
- 🎨 **卡通手繪地圖** — 專屬澳門的遊戲化視覺引擎
- 🎮 **Pokémon GO式探索** — GPS+Wi-Fi指紋輔助，到達自動觸發
- 👴 **長者模式** — 獨立界面，大字+語音，極簡操作
- 📖 **原創IP故事線** — 中葡歷史文化沉浸體驗
- 🤖 **AI智能規劃** — 個性化路線定製（V2.0）

**目標用戶：** 首次來澳旅客 + 深度遊/回頭客

## 核心功能

### 卡通手繪地圖系統
- CDN瓦片加載：縮略圖即時（<100KB）、瓦片按需（12/48片）
- 地圖分層：定位層 → 卡通底圖 → POI標注 → 遊戲化元素 → AR疊加

### GPS+Wi-Fi指紋定位系統
- GPS（室外5-20米）+ Wi-Fi指紋（室內10-30米）+ 手動補簽（200米兜底）
- 動態觸發半徑：30/50/80米

### 遊戲化探索系統
**印章類型：** 足跡印章、故事印章、任務印章、秘密印章、組隊印章（V3.0）

**成就頭銜：**

| 等級 | 頭銜 | 達成條件 |
|------|------|----------|
| 🥉 青銅 | 探索新手 | 完成首個印章 |
| 🥈 白銀 | 澳門見習生 | 10印章 |
| 🥇 黃金 | 澳門達人 | 30印章 |
| 💎 鑽石 | 澳門傳奇 | 60印章 |
| 👑 王者 | 澳門守護者 | 全部完成 |

### 原創IP故事系統（MVP先上2條）
1. 🔮 **海上絲路** — 澳門開埠歷史（6章）
2. ⚔️ **東西方戰事** — 葡澳防衛史（8章）

**原創IP角色：** 小龍、阿蓮、老約翰、蛋撻俠、蓮花娘

### 適老化設計系統
- 長者模式：60x60px觸摸區、20px起字體、語音自動播報、二次確認防誤觸
